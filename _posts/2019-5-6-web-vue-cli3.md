## 简单点

* ubuntu18 安装vue

```
sudo yarn global add @vue/cli
```

* 安装快速原型cli组件

用于快速开发，如单个.vue
```sh
sudo yarn global add @vue/cli-service-global

# a.vue is ready
vue serve src/a.vue
```


* 创建demo
```sh
vue create vue-demo
```
* serve
```sh
cd  vue-demo
yarn serve
```

* webpack配置

touch 1个vue.config.js在根目录.
<https://cli.vuejs.org/zh/config/#vue-config-js>
